# GitHub Profile Searcher

A simple GitHub Profile Search application built using HTML, CSS, and JavaScript.

Users can search for any GitHub profile and view basic information fetched from the GitHub REST API.

New version '/v2' with improved Ui using tailwindcss - not completed yet

## Features

- Search GitHub users by username
- Display profile picture
- Display name and username
- Show follower and following counts
- Show user bio
- Direct link to GitHub profile
- Error handling for invalid usernames
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub REST API
